# 软糯乙女小手机（Web 版）

这是一个使用 React + Vite + TypeScript 构建的「小手机」风格网页应用，主打软糯糯、乙女系 UI。  
初始内置的应用有：

- 微信风格聊天
- 微博风格时间线（支持 AI 生成内容的占位）
- 外卖点单
- 小红书风格卡片流（支持 AI 生成内容的占位）
- 购物应用（支持 AI 生成推荐的占位）

## 本地启动

```bash
npm install
npm run dev
```

浏览器访问 `http://localhost:5173` 即可。  
你也可以在手机浏览器中访问同一局域网下的地址（例如 `http://192.168.1.xx:5173`）进行真机测试。

## 📱 部署到云端（让手机独立运行）

想要在电脑关闭的情况下，在 iOS 手机上独立运行，并且代码修改后自动同步更新？

👉 **查看 [DEPLOY.md](./DEPLOY.md) 获取详细部署指南**

**快速开始（推荐使用 Vercel）：**
1. 将代码推送到 GitHub
2. 访问 [Vercel](https://vercel.com) 并导入仓库
3. 自动部署完成，获得访问链接
4. 在手机上打开链接，添加到主屏幕即可

每次 `git push` 后，应用会自动更新，手机刷新即可看到新版本！
